Change notes


1.0.16

  • Fix Windows path warnings by loading thumbnails from VFS bytes instead of file URLs.

1.0.15

  • Fix SVG thumbnails in Project View by registering FileIconProvider and using IDE ImageLoader for SVG decoding.
  • Run icon providers with order="first" so built-in image icons do not override thumbnails.

1.0.14

  • Upgrade build dependencies (IntelliJ Platform Gradle Plugin, Kotlin, IDE SDK, JSVG).
  • Improve icon loading performance: early extension filtering, VFS reads, concurrent-safe cache, and single-pass scaling.

1.0.13

  • Replace Batik with JSVG for SVG thumbnail rendering, fixing compatibility with IDEA 2026.

1.0.12

  • Fixed the issue of image usage.

1.0.11

  • Fix the bug that prevents SVG from being displayed.
