Image Icon Viewer


Show image thumbnails in the Project View.

SVG icons are rendered with the IDE vector pipeline; raster images use HiDPI-aware scaling when supported.


Supported extensions include those registered with ImageIO.getReaderFileSuffixes()
If you need to add more supported image types:
  1. Download library from: TwelveMonkeys
  2. Put the jar in the lib directory under the IDEA installation directory

Built-in support includes:


  • *.gif
  • *.png
  • *.bmp
  • *.jpg
  • *.jpeg
  • *.webp
  • *.svg
  • plus any format registered with ImageIO at runtime
