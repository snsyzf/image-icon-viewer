Image Icon Viewer


Preview project images right inside the Project View or Tabs.


Supported extensions by ImageIO.getReaderFileSuffixes()
If you need to add more supported image types:
  1. Download library from: TwelveMonkeys
  2. Put the jar in the lib directory under the IDEA installation directory

The default supported extension are:


  • *.gif
  • *.png
  • *.bmp
  • *.jpg
  • *.wbmp
  • *.svg
